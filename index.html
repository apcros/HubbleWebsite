What is Hubble ?

Hubble is a monitoring platform for your servers. It will currently display useful stats like CPU, RAM and Disk usage. It's been made with simplicity and easy setup in mind.

Project evolution

Hubble is a very young project. Because of that it's in constant evolution and loads of things are still missing. Everything, especially API related are likely to change while Hubble is in Alpha. Beta will freeze some of the features and limit some of the evolution. In the meantime you can track the progress here : https://tree.taiga.io/project/apcros-hubble/kanban
